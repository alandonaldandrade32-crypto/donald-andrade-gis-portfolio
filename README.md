## Overview

This portfolio demonstrates my experience in GIS, remote sensing, and spatial data processing, with a focus on environmental and agricultural applications. It includes thematic mapping from a micro-watershed project in India and GIS automation scripts written in Python (ArcPy).

---

## 🗺️ Project: Land Suitability Mapping (Micro-Watershed, India)

### Description
Developed a multi-criteria land suitability model to support watershed-level land management decisions. The study area is a micro-watershed in Karnataka, India.

### Methodology
- Integrated soil type, slope, texture, and terrain depth datasets
- Classified land into suitability categories: S1 (Highly Suitable), S2 (Moderately Suitable), S3 (Marginally Suitable)
- Produced five thematic GIS maps for field reporting

### Tools
ArcGIS Pro

## 🐍 GIS Automation Scripts (Python – ArcPy)

Automated raster processing workflows built with ArcPy. Scripts are in the [`scripts/`](scripts/) folder.

| Script | Description |
|--------|-------------|
| `batch_slope.py` | Batch slope calculation from multiple raster inputs |
| `ndvi_calculation.py` | NDVI computation from multispectral raster bands |
| `clip_rasters.py` | Batch clip rasters to a study area boundary |
